# **App Name**: Asset Tracker

## Core Features:

- Asset Dashboard: Dashboard to display all assets in a sortable table.
- Asset Details: Detailed view for each asset, showing its attributes and history.
- Asset Entry: Simple form to add new assets, capturing key information.
- Reporting: Ability to generate reports on the types, location, and value of assets.
- AI-Powered Asset Categorization: Use an AI tool to automatically categorize new assets based on their descriptions.
- Asset Search: Filtering/searching mechanism for quickly finding specific assets.

## Style Guidelines:

- Primary color: Strong blue (#3498db), evoking trust and stability, which is desirable for an IT application.
- Background color: Light gray (#ecf0f1), offering a clean, neutral backdrop to highlight data.
- Accent color: A vibrant violet (#9b59b6) for interactive elements and highlights, offseting the dominant blue without clashing.
- Body and headline font: 'Inter', a sans-serif font known for its readability.
- Use clear, professional icons to represent different asset types and actions.
- Clean, organized layout with a focus on data presentation and ease of navigation.
- Subtle transitions and animations to provide feedback on user actions and improve overall experience.
