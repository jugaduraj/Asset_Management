# **App Name**: AssetWise

## Core Features:

- Add Asset: Add new hardware or software assets, including details like model, serial number, purchase date, and license information.
- Update Asset Status: Update asset status (e.g., active, inactive, in repair) and ownership details (e.g., assigned user, department).
- View Asset List: Display a comprehensive list of all assets, with search and filter options based on asset type, status, and owner.
- Maintenance Logging: Log and view the maintenance history of each asset, including repair dates, descriptions, and associated costs.
- AI-powered Categorization: Suggest possible categorization of the asset (server, workstation, license...) using asset details using AI; functions as a tool so the user has discretion over how to incorporate its suggestions.

## Style Guidelines:

- Primary color: Dark blue (#2C3E50) to convey professionalism and stability.
- Background color: Light gray (#EEEEEE) for a clean, modern interface; ensures high readability with the dark blue elements.
- Accent color: A slightly brighter blue (#3498DB) is selected to contrast with the primary, to highlight interactive elements and important calls to action.
- Body and headline font: 'Inter', a sans-serif font, for a modern, machined, objective, neutral look; suitable for both headlines and body text.
- Use clean, professional icons to represent asset types, status indicators, and actions.
- Emphasize a clear and organized layout to manage large sets of IT assets. Use consistent spacing, alignment, and visual hierarchy to improve navigation.
