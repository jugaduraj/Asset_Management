# Asset Tracker

This is an IT asset management application built with Next.js.

## Core Features

- **Asset Dashboard**: Display, sort, and search all assets.
- **Asset Details**: A detailed view for each asset.
- **Asset Entry**: A simple form to add new assets.
- **AI-Powered Categorization**: Automatically categorize assets based on their description.
- **Reporting**: Generate reports on asset types and values.
